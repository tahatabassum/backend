## 📝 Folder Document Standard

Each module folder in this repository contains two primary documentation files:
1. **`README.md`**: User guide explaining project overview, directory structure, endpoint documentation, and execution commands.
2. **`Explanation.md`**: In-depth code walkthrough detailing line-by-line function explanation, middleware behavior, and architecture concepts.
